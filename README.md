# Word Race

**Word Race** — обучающая игра для развития навыков быстрого набора текста. Игрок должен вводить движущиеся по экрану слова, зарабатывая очки.

## 🎮 Игровой процесс

- Слова движутся **слева направо** по игровому полю.
- Игрок вводит слово в поле ввода и нажимает **Enter**.
- При правильном вводе:
  - Слово исчезает с экрана
  - Начисляются очки (количество очков = длине слова)
- Если слово достигает **правого края экрана** — оно исчезает, очки **не начисляются**.
- Игра продолжается в течение установленного времени.
- По истечении времени показывается финальный счёт.

## 🎛️ Настройки

| Параметр | Описание |
|----------|----------|
| **Длительность игры** | Время одного раунда (от 10 до 300 секунд) |
| **Скорость слов** | Скорость движения слов (от медленной до очень быстрой) |
| **Язык** | Английский или русский |
| **Список слов** | Загрузка собственного словаря из текстового файла |

### Формат файла со словами

Файл должен быть в кодировке **UTF-8** с расширением `.txt`. Поддерживаются:

- Одно слово на строку
- Слова через запятую в одной строке
- Комментарии (строки, начинающиеся с `#`)
